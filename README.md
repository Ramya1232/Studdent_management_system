# Student Management System

A web-based Student Management System developed using Java, JSP, Servlets, JDBC, MySQL, and Apache Tomcat.

## Features

- User Registration & Login
- Role-Based Login (Admin, HR, Trainer, Student)
- Student CRUD Operations
- HR CRUD Operations
- Trainer CRUD Operations
- Batch Management
- Assign Students to Batches
- Trainer Batch Management
- Requirement Management
- Profile Management

## Technologies Used

- Java
- JSP
- Servlets
- JDBC
- MySQL
- Apache Tomcat
- HTML
- CSS
- JavaScript

## Database

Import `studentmanagementsystem.sql` into MySQL.

Update the database username and password in the JSP files before running the project.
